# My Bus Map
My Bus Map 是一個練習作品，UI設計和功能模仿Google play上知名公車app:

[Bus+](https://play.google.com/store/apps/details?id=hearsilent.busplus&hl=zh_TW&gl=US)

## 功能
- 顯示目前位置附近的公車站牌
- 取得公車站牌的路線資訊、抵達時間
- 搜尋地點，取得該地點附近的公車站牌
- 收藏公車路線

## API
- 公車動態資訊、站牌資訊等使用[TDX運輸資料流通服務](https://tdx.transportdata.tw/)
- Google Maps SDK
